Image buttons

You can include stylish buttons by using a non-standard technique of using an animated gif image with 2 or 3 frames and by setting the active attribute on the image tag.

<a href='../sb-images.html'><img src='../images/image-button.gif' active border='0'></a>

Note the use of the active attribute.

The first frame is used for the normal state, the second frame for the mouse-over state and the (optional) third frame for the mouse-down state.
